# 客户端打包与部署指南

## 📦 打包方案对比

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **EXE打包** | 单文件，即点即用 | 打包时间长，体积大 | 快速部署，不熟悉技术 |
| **便携版Python** | 灵活，易更新 | 需要准备Python | 长期使用，需要更新 |
| **直接部署** | 简单，体积小 | 需目标机有Python | 目标机已有Python |

---

## 🚀 方案一：EXE打包（推荐）

### 步骤

1. **运行打包脚本**
   ```
   双击: 打包客户端.bat
   ```

2. **等待打包完成**
   - 首次运行会自动安装PyInstaller
   - 打包时间约2-5分钟

3. **获取打包文件**
   打包完成后，EXE文件位置：
   ```
   client_package/dist/库存管理客户端.exe
   ```

4. **分发部署**
   将以下文件复制到目标电脑：
   - `库存管理客户端.exe`
   - `inventory_client_config.json` (可选，预配置文件)

### 预配置（可选）

在打包前：
1. 运行配置器配置好客户端
2. 将生成的 `inventory_client_config.json` 一起复制

---

## 🚀 方案二：便携版部署（灵活）

### 步骤

1. **创建便携版部署包**
   ```
   双击: 创建便携版客户端.bat
   ```

2. **准备便携版Python（可选）**

   **方法A：嵌入式Python（推荐）**
   - 下载地址: https://www.python.org/downloads/windows/
   - 下载: Windows embeddable package (64-bit)
   - 解压到: `client_portable/python_portable/`
   - 下载: https://bootstrap.pypa.io/get-pip.py
   - 运行: `python_portable\python.exe get-pip.py`
   - 运行: `安装依赖.bat`

   **方法B：WinPython**
   - 下载地址: https://winpython.github.io/
   - 下载: WinPython Zero
   - 解压到: `client_portable/python_portable/`

3. **添加预配置（可选）**
   - 从配置器导出配置
   - 保存为: `client_portable/inventory_client_config.json`

4. **部署到目标电脑**
   - 将整个 `client_portable` 文件夹复制到U盘
   - 复制到目标电脑任意位置
   - 双击 `启动客户端.bat`

---

## 🚀 方案三：直接部署（最简单）

### 步骤

1. **创建部署包**
   ```
   双击: 创建客户端部署包.bat
   ```

2. **部署到目标电脑**
   - 复制整个 `client_deploy` 文件夹
   - 确保目标电脑已安装Python 3.8+
   - 运行 `安装依赖.bat`
   - 运行 `启动客户端.bat`

---

## 📋 预配置客户端

### 方法：使用配置器导出

1. **在服务器电脑上运行配置器**
   ```
   双击: 启动配置器.bat
   ```

2. **配置客户端**
   - 服务器地址: 填写服务器的局域网IP
   - API密钥: 与服务器一致
   - 其他选项按需配置

3. **导出配置**
   - 点击「导出客户端配置」
   - 保存为: `inventory_client_config.json`

4. **将配置文件一起打包**

## 🔧 常见问题

### Q: PyInstaller打包失败？

A: 检查：
1. Python版本是否为3.8+
2. 是否有足够的磁盘空间
3. 临时文件夹权限
4. 尝试以管理员身份运行

### Q: EXE文件太大？

A: 正常现象，Python打包会包含解释器和依赖库。
建议使用便携版方案。

### Q: 如何减小EXE体积？

A: 可以：
1. 使用UPX压缩（PyInstaller默认使用）
2. 排除不必要的模块
3. 使用虚拟环境纯净打包

### Q: 目标电脑没有Python？

A: 使用方案一（EXE）或方案二（便携版Python）

### Q: 如何更新客户端？

A: 
- EXE版：重新打包分发
- 便携版：替换 inventory_client.py
- 直接版：替换 inventory_client.py

### Q: 客户端无法连接服务器？

A: 检查：
1. 服务器已启动
2. IP地址正确（局域网IP）
3. API密钥一致
4. 防火墙开放端口
5. 两台电脑在同一网络

## 💡 推荐使用场景

### 场景1：快速部署给非技术人员
**推荐：方案一（EXE打包）**
- 只需一个EXE文件
- 即点即用，无需配置环境

### 场景2：需要频繁更新
**推荐：方案二（便携版）**
- 只需替换主程序文件
- 便于维护和更新

### 场景3：目标电脑已有Python
**推荐：方案三（直接部署）**
- 最省空间
- 最简单快速

## 🔐 安全建议

1. **API密钥保护**
   - 预配置文件包含API密钥
   - 仅在可信环境分发
   - 定期更换API密钥

2. **网络安全**
   - 建议在局域网内使用
   - 如需公网访问使用VPN
   - 考虑启用HTTPS

3. **数据安全**
   - 客户端不存储业务数据
   - 数据都在服务器数据库
   - 定期备份服务器数据
