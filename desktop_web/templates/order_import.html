📥 订单批量导入

上传标准 Excel 模板，批量创建订单（支持 .xlsx / .xls）

1下载标准模板
点击上方「下载标准模板」按钮，获取标准 Excel 文件。模板包含18个字段，其中 客户名称、产品类型、数量 为必填，其余选填。
2填写 Excel
按行填写订单数据，支持多个订单（从第2行开始，第1行是表头勿修改）。必填字段：客户名称、产品类型、数量。
必填 客户名称 必填 产品类型 必填 数量 选填 订单号 选填 客户分组 选填 客户电话 选填 客户地址 选填 材质 选填 网孔尺寸 选填 丝径(mm) 选填 宽度(mm) 选填 长度(m) 选填 单位 选填 单价(元) 选填 表面处理 选填 特殊要求 选填 交期(YYYY-MM-DD) 选填 备注
3上传并导入
上传填写好的 Excel 文件，系统将自动解析并导入。已存在的订单号会自动跳过。
📁
拖拽 Excel 文件到这里，或 点击选择文件
支持 .xlsx / .xls，最大 10MB
