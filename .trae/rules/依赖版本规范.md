# 依赖版本规范

## 环境要求

### Python 版本
- **固定版本**: Python 3.14.3
- **可执行文件路径**: `C:\Users\lenovo\AppData\Local\Python\bin\python.exe`
- **所有命令必须使用此路径**，禁止使用 `python`、`python3`、`py -3` 等其他调用方式

### 命令调用规范

```powershell
# ✅ 正确
& "C:\Users\lenovo\AppData\Local\Python\bin\python.exe" script.py

# ❌ 错误
python script.py
py -3 script.py
```

### 重启服务规范

使用 Python 脚本管理进程，禁止在 PowerShell 命令中混用多个 `&` 操作符：

```python
# ✅ 正确：用 Python 脚本管理进程
import subprocess

# 杀进程
for pid in [123, 456]:
    subprocess.run(["taskkill", "/PID", str(pid), "/F"])

# 启动服务
subprocess.Popen(
    [PYTHON, script_path],
    cwd=WORK_DIR,
    creationflags=subprocess.CREATE_NEW_CONSOLE
)
```

```powershell
# ❌ 错误：PowerShell 不支持 & 连接符
taskkill /PID 123 /F & taskkill /PID 456 /F
```
