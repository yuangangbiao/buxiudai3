### 阶段三：CI/CD集成（6月25日-6月30日）

#### 任务 3.1：创建GitHub Actions配置

**文件**：
```yaml
# .github/workflows/test.yml
name: Test Suite

on:
  push:
    branches: [main, develop]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.12'
      
      - name: Install dependencies
        run: |
          pip install pytest pytest-cov pytest-html
      
      - name: Run tests
        run: |
          pytest tests/unit --cov=. --cov-report=xml --html=reports/test_report.html
      
      - name: Upload coverage
        uses: codecov/codecov-action@v3
        with:
          file: ./coverage.xml
      
      - name: Upload test report
        uses: actions/upload-artifact@v3
        with:
          name: test-report
          path: reports/test_report.html
```

#### 任务 3.2：创建本地CI脚本

**文件**：`tests/run_ci.py`

```python
#!/usr/bin/env python
"""本地CI脚本 - 运行完整测试套件"""
import subprocess
import sys
from pathlib import Path
from datetime import datetime

def run_cmd(cmd, cwd=None):
    """执行命令"""
    print(f"\n{'='*60}")
    print(f"执行: {cmd}")
    print('='*60)
    result = subprocess.run(cmd, shell=True, cwd=cwd)
    return result.returncode == 0

def main():
    root = Path(__file__).parent.parent
    
    # 1. 收集测试用例
    if not run_cmd("python .workbuddy/tools/regression_db.py import-tests", root):
        print("[FAIL] 测试用例收集失败")
        sys.exit(1)
    
    # 2. 运行测试
    if not run_cmd("pytest tests/unit --cov=. --cov-report=term --tb=short", root):
        print("[FAIL] 测试运行失败")
        sys.exit(1)
    
    # 3. 生成报告
    run_cmd("pytest tests/unit --html=reports/test_report.html --self-contained-html", root)
    
    print("\n✅ CI流程完成!")

if __name__ == '__main__':
    main()
```

## 四、风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|---------|
| 测试执行慢 | 开发反馈延迟 | 使用 pytest-xdist 并行执行 |
| Mock过度使用 | 测试与实际不符 | 添加集成测试 |
| 测试覆盖不均 | 遗漏关键路径 | 添加代码覆盖率检查 |
| 数据库污染 | 测试结果不稳定 | 使用事务回滚 |
