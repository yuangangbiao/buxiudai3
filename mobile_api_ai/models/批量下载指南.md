对于人脸考勤系统，**强烈推荐使用 buffalo_l 模型**：


## 常见问题

### 1. GitHub下载速度慢
**解决方案**：
- 使用代理或VPN
- 使用国内镜像：
  ```
  https://ghproxy.com/https://github.com/deepinsight/insightface/releases/download/v0.7/buffalo_l.zip
  ```
- 使用迅雷等下载工具

### 2. 解压失败
**解决方案**：
- 使用7-Zip解压：https://www.7-zip.org/
- 确保磁盘空间充足（需要解压后2倍空间）

### 3. 模型路径问题
**解决方案**：
- 确保模型解压到正确目录
- 或设置环境变量：
  ```bash
  set INSIGHTFACE_MODEL_DIR=d:\yuan\不锈钢网带跟单3.0\mobile_api_ai\models
  ```
