# INSIGHTFACE buffalo_l 模型手动下载指南

## 方法一：浏览器直接下载（推荐）

1. 打开以下链接下载压缩包（约160MB）：
   ```
   https://github.com/deepinsight/insightface/releases/download/v0.7/buffalo_l.zip
   ```

2. 将下载的 `buffalo_l.zip` 移动到项目目录：
   ```
   d:\yuan\不锈钢网带跟单3.0\mobile_api_ai\models\buffalo_l\
   ```

3. 解压文件到当前目录

4. 验证文件结构：
   ```
   models\buffalo_l\
   ├── buffalo_l.zip (可选，删除以节省空间)
   ├── buffalo_l.json
   ├── det_10g.onnx
   ├── w600k_r50.onnx
   └── 2d106det\
       └── 2d106det.onnx
   ```

## 方法二：使用下载工具

### IDM (Internet Download Manager)
1. 新建任务
2. URL: `https://github.com/deepinsight/insightface/releases/download/v0.7/buffalo_l.zip`
3. 保存到: `d:\yuan\不锈钢网带跟单3.0\mobile_api_ai\models\buffalo_l\`
4. 开始下载

### 迅雷
1. 新建任务
2. 粘贴链接: `https://github.com/deepinsight/insightface/releases/download/v0.7/buffalo_l.zip`
3. 选择保存路径
4. 开始下载

## 方法三：使用Python脚本

双击运行以下脚本：

d:\yuan\不锈钢网带跟单3.0\mobile_api_ai\scripts\download_model.py

## 常见问题

### GitHub下载失败
由于网络原因，GitHub直连可能较慢。可尝试：
- 使用加速镜像：https://ghproxy.com/
  ```
  https://ghproxy.com/https://github.com/deepinsight/insightface/releases/download/v0.7/buffalo_l.zip
  ```
- 使用代理/VPN
- 使用国内镜像（如有）

### 解压失败
使用以下工具解压：
- 7-Zip（推荐）：https://www.7-zip.org/
- WinRAR
- Windows自带解压工具
