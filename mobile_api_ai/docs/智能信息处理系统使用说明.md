# 不锈钢网带跟单系统 - 智能信息处理系统

## 📋 系统概述

本系统实现了完整的智能指令交互处理，包括：

- **容器中心**: 作为数据传输中心，管理任务流转
- **企业微信机器人**: 员工在群里报工、查询进度
- **桌面端集成**: 跟单系统自动发布任务、接收回调
- **小程序**: 移动端操作（可选）

## 🚀 快速开始

### 1. 环境准备

确保已安装Python 3.8+，然后安装依赖：

```bash
cd mobile_api_ai
pip install -r requirements.txt
```

### 2. 配置文件

创建 `.env` 文件（已为您创建）：

```env
# 企业微信机器人Webhook
WECHAT_WORK_BOT_URL=https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=...

# 容器中心地址
CONTAINER_CENTER_URL=http://localhost:5002

# AI模式 (mock/production)
AI_MODE=mock
```

### 3. 启动服务

方式一：使用统一启动脚本（推荐）

```bash
python start_all.py
```

方式二：分别启动

```bash
# 终端1: 启动容器中心
python container_api_server.py

# 终端2: 启动企业微信机器人
python wechat_work_bot_v2.py
```

### 4. 测试集成

```bash
python test_integration.py
```

### 桌面端集成使用

在跟单系统中引入桌面客户端：

```python
from desktop_container_client import ContainerPoolClient, AutoPublishHelper

# 创建客户端
client = ContainerPoolClient('http://localhost:5002')

# 登录
client.login('OP001')

# 当工序完成时自动发布报工任务
def on_process_finished(order_no, process_name, record_id, operator_id, qty):
    client.publish_report_task(
        order_no=order_no,
        process_name=process_name,
        record_id=record_id,
        operator_id=operator_id,
        planned_qty=qty
    )

# 自动发布助手
helper = AutoPublishHelper('http://localhost:5002')

# 工序完成时
helper.on_process_completed('ORD202604001', '编织', 103, 'OP001', 100)

# 需要质检时
helper.on_quality_required('ORD202604001', 1, 'OP004', '终检')

# 物料需求时
helper.on_material_needed('ORD202604001', '不锈钢丝', 50, 'OP003')
```

### 桌面端回调监听

桌面端可以监听容器中心的回调：

```python
from integration import DesktopClientListener

class MyDesktopListener(DesktopClientListener):
    def __init__(self):
        super().__init__('http://localhost:8888/callback')
    
    def on_task_assigned(self, data):
        """新任务分配时"""
        print(f'收到新任务: {data.get("task_title")}')
    
    def on_task_completed(self, data):
        """任务完成时"""
        print(f'任务完成: {data.get("task_title")}')
    
    def on_data_collected(self, data):
        """数据收集时"""
        print(f'数据已收集: {data.get("data_type")}')

# 启动监听
listener = MyDesktopListener()
```

## 📊 API接口文档

### 容器中心 API (端口 5002)

#### 认证

```http
POST /api/auth/login
Content-Type: application/json

{
  "operator_id": "OP001"
}
```

#### 发布任务（内部接口）

```http
POST /api/internal/publish
Authorization: Bearer <token>
Content-Type: application/json

{
  "task_type": "report",
  "title": "报工：编织",
  "content": {
    "record_id": 103,
    "planned_qty": 100,
    "order_no": "ORD202604001"
  },
  "operator_id": "OP001",
  "priority": "normal",
  "related_order": "ORD202604001",
  "related_process": "编织"
}
```

#### 查询任务列表

```http
GET /api/pool/tasks/pending
Authorization: Bearer <token>
```

#### 获取池状态

```http
GET /api/pool/status
Authorization: Bearer <token>
```

### 企业微信机器人 API (端口 5003)

#### 获取机器人状态

```http
GET /api/wechat/status
```

#### 查询订单任务

```http
GET /api/wechat/pool/tasks/<order_no>
```

#### 获取容器池状态

```http
GET /api/wechat/pool/status
```

#### 测试发送消息

```http
GET /api/wechat/test
```

## 🔧 任务类型支持

### 1. 报工任务 (report)

- 工序完成报工
- 支持语音和文字报工
- 自动分配给对应操作员

### 2. 质检任务 (quality)

- 订单质检
- 支持拍照分析
- 质检员处理

### 3. 物料需求 (material)

- 物料领料申请
- 库存检查
- 物料员处理

### 4. 审批任务 (approval)

- 异常情况审批
- 经理确认
- 高优先级处理

### 常见问题

**Q: 企业微信群没有收到通知？**

A: 检查 `.env` 文件中的 `WECHAT_WORK_BOT_URL` 配置是否正确。

**Q: 桌面端收不到回调？**

A: 确认桌面端已注册回调，且回调URL可访问。

**Q: 任务状态没有更新？**

A: 检查容器中心服务是否正常运行，SQLite数据库权限是否正确。
