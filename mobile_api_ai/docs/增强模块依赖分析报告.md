## 二、详细依赖清单

### 2.1 P0 - 核心依赖（必须安装）

| 依赖包 | 版本 | 用途 | 模块依赖 | 状态 |
|--------|------|------|----------|------|
| `redis` | >=4.5.0 | Redis客户端 | 队列管理、熔断器、缓存 | ❌ 缺失 |
| `elasticsearch` | >=8.0.0 | ES客户端 | 审计日志存储 | ❌ 缺失 |
| `psutil` | >=5.9.0 | 系统监控 | 健康检查、资源监控 | ❌ 缺失 |

**影响分析：**
- 没有Redis：队列管理、熔断器、缓存功能无法使用
- 没有Elasticsearch：审计日志无法存储和检索
- 没有psutil：系统健康检查功能受限

| 依赖包 | 版本 | 用途 | 模块依赖 | 状态 |
|--------|------|------|----------|------|
| `ntplib` | >=0.4.0 | NTP时间同步 | 时钟同步服务 | ❌ 缺失 |
| `pywin32` | >=306 | Windows系统时间 | 时钟同步（Windows） | ❌ 缺失 |

**影响分析：**
- 没有ntplib：时钟同步功能受限，数据漂移检测不准确
- 没有pywin32：Windows系统时间无法自动校正

## 四、依赖安装指南

### 4.1 快速安装（推荐）

```bash
# 1. 安装Python依赖
pip install -r enhanced_requirements.txt

# 2. 安装系统服务（Ubuntu/Debian）
sudo apt update
sudo apt install redis-server elasticsearch ntp

# 3. 启动服务
sudo systemctl enable redis-server elasticsearch ntp
sudo systemctl start redis-server elasticsearch ntp
```

### 4.2 分步安装（按需）

#### 仅安装核心依赖
```bash
pip install redis>=4.5.0 elasticsearch>=8.0.0 psutil>=5.9.0
```

#### 安装完整增强依赖
```bash
pip install -r enhanced_requirements.txt
```

## 六、常见问题解决

### 6.1 Redis连接问题

**问题：** Redis服务无法连接
**解决：**
```bash
# 检查Redis服务状态
sudo systemctl status redis-server

# 修改Redis配置允许远程连接
sudo nano /etc/redis/redis.conf
# 修改: bind 127.0.0.1 → bind 0.0.0.0

# 重启Redis服务
sudo systemctl restart redis-server
```

### 6.2 Elasticsearch内存不足

**问题：** Elasticsearch启动失败
**解决：**
```bash
# 修改JVM内存设置
sudo nano /etc/elasticsearch/jvm.options
# 修改: -Xms1g -Xmx1g

# 重启Elasticsearch
sudo systemctl restart elasticsearch
```

### 6.3 时间同步失败

**问题：** NTP同步失败
**解决：**
```bash
# 配置阿里云NTP服务器
sudo nano /etc/ntp.conf
# 添加: server ntp.aliyun.com iburst

# 重启NTP服务
sudo systemctl restart ntp
```
