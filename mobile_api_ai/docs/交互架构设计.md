## 🔄 指令流转核心流程

### 流程一：企业微信群报工 → 桌面系统

```
步骤1：员工在群里发消息
│
▼
[企业微信机器人] 解析报工指令
│  "报工 ORD202604001 编织 200"
│
▼
[容器中心] 收集任务并分配
│  任务ID: TASK202604300001
│  状态: pending → distributed
│
▼
[容器中心] 分发给指定操作员
│  同时记录到SQLite数据库
│
▼
[桌面系统] 轮询或接收推送
│  桌面端通过回调API监听容器池变化
│
▼
[桌面系统] 更新工序记录
│  将报工数据写入跟单系统数据库
│
▼
[容器中心] 标记任务完成
│  状态: distributed → completed
│
▼
[企业微信机器人] 群里反馈
│  "[OK] 报工成功！订单 ORD202604001..."
```

### 流程二：桌面系统 → 容器池 → 企业微信群通知

```
步骤1：桌面系统工序完成
│
▼
[桌面客户端] 自动发布任务到容器池
│  client.publish_report_task(...)
│
▼
[容器中心] 分配任务给相关人员
│
▼
[企业微信机器人] 发送群通知
│  "@张三 您有新的质检任务: ORD202604001"
│
▼
员工响应并处理
│
▼
[容器中心] 更新任务状态
│
▼
[桌面系统] 轮询获取更新
```

## 📡 通信接口设计

### 1. 容器中心 API (端口 5002)

```
内部发布接口:
  POST /api/internal/publish
  {
    "task_type": "report",  // report/quality/material/approval
    "title": "报工：编织",
    "content": {...},
    "operator_id": "OP001",
    "priority": "normal",
    "related_order": "ORD202604001",
    "related_process": "编织"
  }

任务状态查询:
  GET /api/pool/status
  GET /api/pool/tasks/{status}

任务更新回调:
  POST /api/internal/callback/{task_id}
  {
    "status": "completed",
    "result": {...},
    "completed_at": "2026-04-30T10:00:00"
  }
```

### 2. 企业微信机器人 API (端口 5003)

```
接收群消息:
  POST /api/wechat/hook
  (企业微信回调格式)

发送群消息:
  内部调用企业微信Webhook

机器人状态:
  GET /api/wechat/status

测试发送:
  GET /api/wechat/test
```

### 3. 桌面端回调服务

```
桌面端监听接口 (可选):
  POST /desktop/callback
  {
    "event": "task_assigned",
    "task_id": "TASK202604300001",
    "data": {...}
  }
```

## 🎯 核心交互场景

### 场景1：员工在群里报工
```
1. 员工: "报工 ORD202604001 编织 200"
2. 机器人: 解析 → 发布任务到容器池
3. 容器池: 存储 → 分配
4. 桌面系统: 轮询检测到新任务
5. 桌面系统: 自动更新工序记录
6. 容器池: 标记完成
7. 机器人: "[OK] 报工成功！"
```

### 场景2：桌面端自动发布质检任务
```
1. 桌面系统: 编织工序完成
2. 桌面客户端: publish_quality_task(...)
3. 容器池: 存储任务
4. 机器人: "@质检员 您有新的质检任务"
5. 质检员: 拍照上传质检结果
6. 容器池: 分析数据
7. 桌面系统: 回调接收质检结果
8. 机器人: "质检完成，质量合格"
```

### 场景3：查询订单进度
```
1. 员工: "查单 ORD202604001"
2. 机器人: 查询容器池
3. 容器池: 获取相关任务列表
4. 机器人: 格式化回复
5. 群里显示: 订单进度信息
```

## 🚀 启动顺序

1. **容器中心服务** (端口 5002) - 先启动
2. **企业微信机器人** (端口 5003) - 后启动
3. **桌面跟单系统** - 最后启动，连接容器池
