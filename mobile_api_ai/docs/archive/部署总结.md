## 🔧 常见问题快速参考

### 问题1：连接超时
**原因**：5003端口未开放  
**解决**：配置腾讯云安全组 + 服务器防火墙

### 问题2：无法验证URL
**原因**：校验文件无法访问或服务未启动  
**解决**：检查服务状态，验证校验文件

### 问题3：pip安装失败
**原因**：网络问题  
**解决**：使用国内镜像源（`-i https://pypi.tuna.tsinghua.edu.cn/simple`）

### 问题4：Python命令不识别
**原因**：未添加到PATH  
**解决**：重新安装Python，勾选"Add Python to PATH"

**更多问题**：请参考 `完整部署指南.md` 的"常见问题"章节
