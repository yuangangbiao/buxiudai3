## 二、部署步骤

### 步骤1：远程连接服务器

1. 在本地电脑按 `Win + R`，输入 `mstsc`，回车

## 四、常见问题处理

### Q1：提示"连接超时"

**原因**：5003端口未开放

**解决方法**：
1. 检查腾讯云安全组规则，添加入站规则：
   - 协议：TCP
   - 端口：5003
   - 来源：0.0.0.0/0

2. 检查服务器防火墙，开放5003端口：
   ```cmd
   netsh advfirewall firewall add rule name="WeChatBot" dir=in action=allow protocol=TCP localport=5003
   ```

---

### Q2：提示"无法验证URL"

**原因**：校验文件无法访问或服务未启动

**解决方法**：
1. 确认服务正在运行
2. 浏览器访问：`http://124.223.57.82:5003/WW_verify_PWFveCpOUtSmyNnB.txt`
3. 如果能访问，检查文件内容是否为 `PWFveCpOUtSmyNnB`
4. 如果无法访问，检查防火墙和安全组

---

### Q3：pip安装失败

**原因**：网络问题或权限不足

**解决方法**：
1. 使用管理员权限运行CMD
2. 使用国内镜像源：
   ```cmd
   python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```
3. 升级pip：
   ```cmd
   python -m pip install --upgrade pip
   ```

---

### Q4："python不是内部或外部命令"

**原因**：Python未添加到PATH

**解决方法**：
1. 重新安装Python，务必勾选"Add Python to PATH"
2. 或手动添加环境变量：
   - 右键"此电脑" → "属性" → "高级系统设置" → "环境变量"
   - 在"系统变量"中找到"Path"，点击"编辑"
   - 添加Python安装路径（如 `C:\Python311\` 和 `C:\Python311\Scripts\`）

---

### Q5：服务启动后立即退出

**原因**：代码错误或端口被占用

**解决方法**：
1. 查看错误信息，检查 `.env` 配置是否正确
2. 检查5003端口是否被占用：
   ```cmd
   netstat -ano | findstr :5003
   ```
3. 如果端口被占用，修改 `.env` 中的 `FLASK_PORT`，或停止占用端口的程序

---

### Q6：企业微信收不到消息

**原因**：消息发送失败或配置错误

**解决方法**：
1. 检查 `.env` 中的企业微信配置是否正确：
   - `WECHAT_CORP_ID`
   - `WECHAT_AGENT_ID`
   - `WECHAT_SECRET`
2. 查看服务日志，检查错误信息
3. 手动测试发送消息：
   ```cmd
   curl http://124.223.57.82:5003/api/wechat/test
   ```
