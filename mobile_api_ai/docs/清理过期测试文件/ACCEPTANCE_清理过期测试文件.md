## 教训（Lessons Learned）

1. **dry-run 必须覆盖全部目录**：不能只核查 mobile_api_ai/，必须遍历整个项目
2. **按子目录白名单保护**：对按日期命名的子目录（如 e2e_20260623/）应自动加入白名单
3. **分级清理流程**：大批量删除必须分批执行，每批之间插入核验点
4. **运行中服务优先**：清理前应先识别运行中服务，自动排除其日志目录
5. **PowerShell 性能问题**：75k+ 文件删除必须用 .NET 编译方式（.NET 删除 21,343 个用了 493 秒，平均 23ms/个；PowerShell Get-ChildItem + Remove-Item 慢 10 倍以上）
