### 3.1 过期基准日

### 3.2 清理原则

| 类别 | 策略 |
|------|------|
| 修改时间 ≥ 2026-06-13 | ✅ 全部保留 |
| 修改时间 < 2026-06-13 | 🗑️ 全部清理（除业务关键日志） |

| 0 字节空 .log | 🗑️ 全部清理 |

- **黑名单保护**：
  - `mobile_api_ai/modules/DAT/audit_log/`（业务审计）

  - `mobile_api_ai/logs/2026-06-13 之后所有日志`
- **白名单保护**：
  - `scripts/test_ux_xiaoxi.log`（用户测试输出）
  - `.trae/logs/*.log`（项目级日志）
  - `.coveragerc`（配置）
