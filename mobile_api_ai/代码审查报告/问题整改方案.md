## P0-2: 裸 `except:` 修复详情

### 修复原则
```
# ❌ 错误写法
try:
    result = some_function()
except:
    pass

# ✅ 正确写法  
try:
    result = some_function()
except Exception as e:
    logger.error(f"操作失败: {e}")

# ✅ 带返退的写法
try:
    result = some_function()
except Exception as e:
    logger.error(f"操作失败: {e}")
    return default_value
```

## P1-3: `print()` → `logger` 替换

### 修复原则
```
# ❌ 错误 (生产代码)
print(f"[SQLite] 保存数据包失败: {e}")

# ✅ 正确
logger.error(f"SQLite保存数据包失败: {e}")
```

### 日志级别使用规范
| 场景 | 级别 | 示例 |
|------|------|------|
| 正常操作 | `logger.info()` | 服务启动、连接成功 |
| 可恢复错误 | `logger.warning()` | 连接失败回退、重试 |
| 不可恢复错误 | `logger.error()` | 操作失败、异常 |
| 调试信息 | `logger.debug()` | 函数入口/出口 |
