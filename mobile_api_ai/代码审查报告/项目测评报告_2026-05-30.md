## 3. 详细发现

### 3.1 [CRITICAL] 硬编码密码

**位置**：
- `scripts/fix_production_orders.py:25` - `'88888888'`
- `scripts/check_process_records.py:17` - `'88888888'`
- `scripts/tools/sync_container_to_cs.py:97` - `'cs123456'`

**风险**：敏感信息泄露至版本控制

**修复**：
```python
# 错误
'password': '88888888'

# 正确
'password': os.environ.get('MYSQL_PASSWORD', '')
```

### 3.2 [HIGH] 硬编码 secret_key

**位置**：
- `inventory_api_server.py:18` - `'inventory_web_secret_2026'`

**风险**：JWT 签名密钥可被外部猜测，威胁认证安全

**修复**：
```python
# 错误
app.secret_key = os.getenv('FLASK_SECRET_KEY', 'inventory_web_secret_2026')

# 正确（不提供默认值）
app.secret_key = os.getenv('FLASK_SECRET_KEY')
```

### 3.3 [HIGH] 裸 except 语句

**统计**：20+ 处

**位置**：
- `dispatch_center.py`: 8 处
- `container_center_api.py`: 2 处
- `api/legacy_routes.py`: 1 处
- `storage_layer.py`: 2 处
- `scripts/tools/*.py`: 多处

**示例**：
```python
# 错误
try:
    ...
except: pass

# 正确
try:
    ...
except Exception as e:
    logger.exception(f"操作失败: {e}")
```
