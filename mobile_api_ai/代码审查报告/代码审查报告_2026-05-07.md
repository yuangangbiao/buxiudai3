## 8. 修复示例

### 8.1 裸 `except:` 修复模式
```python
# ❌ 错误
try:
    result = some_function()
except:
    pass

# ✅ 正确
try:
    result = some_function()
except Exception as e:
    logger.error(f"some_function 执行失败: {e}")
    result = None  # 或合适的默认值
```

### 8.2 `debug=True` 修复
```python
# ❌ 错误
app.run(host='0.0.0.0', port=5003, debug=True)

# ✅ 正确
app.run(host='0.0.0.0', port=int(os.getenv('FLASK_PORT', '5003')), debug=False)
```

### 8.3 端口统一管理
在 `config.py` 中添加：
```python
FLASK_PORT = int(os.getenv('FLASK_PORT', '5003'))
```
所有文件引用 `from config import FLASK_PORT`。
