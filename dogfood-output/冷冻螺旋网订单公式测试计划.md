## 六、公式计算详细过程

### 6.1 编制左旋/右旋

```
公式: 总长度 / 网带节距 / 2
代入: 1000 / 16.93333 / 2
计算: 1000 ÷ 16.93333 = 59.0536
     59.0536 ÷ 2 = 29.5268
向上取整: ceil(29.5268) = 30
结果: 30
```

### 6.2 其他工序（除以1000）

```
公式: 总长度 / 1000
代入: 1000 / 1000
计算: 1.0
向上取整: ceil(1.0) = 1
结果: 1
```

## 七、注意事项

1. **订单号**: 用户提到的 `20260420001` 不存在，使用 `ORD-202605010001` 代替（参数最完整）
2. **网带节距**: 16.93333mm 是关键参数，用于编制工序计算
3. **向上取整**: 所有公式结果都经过 `math.ceil()` 向上取整
4. **参数匹配**: 公式中的 `{总长度}` 对应 extra_params.总长度
