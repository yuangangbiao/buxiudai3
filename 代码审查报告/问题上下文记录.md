### 2. eval代码注入模式 ⚠️ 禁止

**错误模式**:
```python
# ❌ 直接eval用户输入
result = eval(condition, {"__builtins__": {}}, self.OPERATORS)
result = eval(formula_str)
```

**正确模式**:
```python
# ✅ 使用安全解析器
import ast

def safe_eval_expr(expr: str) -> Any:
    allowed_chars = set("0123456789.+-*/() ><!=andor_abcdefghijklmnopqrstuvwxyz ")
    if not all(c in allowed_chars for c in expr):
        raise ValueError("非法字符")
    node = ast.parse(expr, mode='eval')
    return eval_ast(node)  # 自定义安全求值
```

### 3. 资源泄露模式 ⚠️ 禁止

**错误模式**:
```python
# ❌ 重复关闭cursor
cursor.execute(...)
cursor.close()
cursor = conn.cursor()
cursor.execute(...)  # cursor已关闭
cursor.close()  # 重复关闭

# ❌ 异常时不关闭连接
conn = get_connection()
cursor = conn.cursor()
cursor.execute(...)  # 如果这里抛异常，conn不会关闭
```

**正确模式**:
```python
# ✅ 每次操作后关闭cursor
cursor.close()
cursor = conn.cursor()  # 重新获取

# ✅ 使用try-finally
conn = get_connection()
try:
    cursor = conn.cursor()
    # ... 操作 ...
finally:
    if cursor:
        cursor.close()
    conn.close()
```

**涉及文件**:
- models/order.py

### 4. 线程安全模式 ⚠️ 禁止

**错误模式**:
```python
# ❌ 全局变量无锁保护
global WECHAT_SECRET
WECHAT_SECRET = os.getenv('WECHAT_SECRET')

# ❌ 共享状态无锁
self._cache = {}
```

**正确模式**:
```python
# ✅ 使用线程锁
import threading

class ConfigManager:
    _lock = threading.Lock()
    _config = {}

    @classmethod
    def set(cls, key, value):
        with cls._lock:
            cls._config[key] = value

    @classmethod
    def get(cls, key):
        with cls._lock:
            return cls._config.get(key)
```

- mobile_api_ai/wechat_work_bot_v2.py
- models/database.py

### 5. 数值类型未校验模式 ⚠️ 禁止

**错误模式**:
```python
# ❌ 直接转换
qty = float(data.get("quantity", 1) or 0)  # "abc"会抛异常
```

**正确模式**:
```python
# ✅ 安全转换
def safe_float(value, default=0.0):
    try:
        return float(value) if value else default
    except (ValueError, TypeError):
        return default

qty = safe_float(data.get("quantity"), 1.0)
```

except Exception as e:

## 编码规范提醒

### 必须使用环境变量管理敏感配置
```
敏感信息 → .env文件 → os.getenv()读取 → 无默认值
```

### 必须使用context manager管理资源
```python
with get_db_cursor() as (cursor, conn):
    # 操作
# 自动关闭
```

### 必须使用参数化查询
```python
cursor.execute("SELECT * FROM users WHERE id=%s", (user_id,))
```

### 必须使用日志记录异常
```python
logger.exception(f"操作失败: {e}")
```
