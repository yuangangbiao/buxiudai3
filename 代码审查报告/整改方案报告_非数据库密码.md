### P1-1: eval代码注入 (task_trigger_engine.py) 🟠 HIGH

**问题文件**: mobile_api_ai/task_trigger_engine.py 第86, 123行
**错误代码**: ERR-SEC-003

**问题代码**:
```python
result = eval(condition, {"__builtins__": {}, "__name__": None}, allowed_ops)  # ❌
result = eval(formula, {"__builtins__": {}, "__name__": None}, allowed_ops)  # ❌
```

**整改方案**:
```python
import ast
from operator import add, sub, mul, truediv, gt, lt, ge, le, eq, ne

SAFE_OPS = {
    '+': add, '-': sub, '*': mul, '/': truediv,
    '>': gt, '<': lt, '>=': ge, '<=': le, '==': eq, '!=': ne,
}

def safe_eval_expr(expr: str):
    """安全表达式求值"""
    if not expr or not isinstance(expr, str):
        return None

    allowed_chars = set("0123456789.+-*/() ><!=andor_abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ ")
    if not all(c in allowed_chars for c in expr.strip()):
        raise ValueError("表达式包含非法字符")

    try:
        node = ast.parse(expr.strip(), mode='eval')
        return eval_ast(node)
    except Exception:
        return None

def eval_ast(node):
    """遍历AST节点安全求值"""
    if isinstance(node, ast.Constant):
        return node.value
    elif isinstance(node, ast.BinOp):
        left = eval_ast(node.left)
        right = eval_ast(node.right)
        op_map = {'Add': add, 'Sub': sub, 'Mult': mul, 'Div': truediv,
                  'Gt': gt, 'Lt': lt, 'GtE': ge, 'LtE': le, 'Eq': eq, 'NotEq': ne}
        op = op_map.get(type(node.op).__name__)
        if op:
            return op(left, right)
    return None
```

**验证方法**: 输入恶意代码 `__import__('os').system('rm -rf')` 不执行

def safe_eval_formula(formula: str) -> float:

    if not formula:

    formula = formula.replace('×', '*').replace('÷', '/').replace('X', '*').replace('x', '*')

    allowed_chars = set("0123456789.+-*/() ")

### P1-6: 全局变量无锁保护 🟠 HIGH

**问题文件**: mobile_api_ai/wechat_work_bot_v2.py 第49-50行
**错误代码**: ERR-BIZ-009

**问题代码**:
```python
global WECHAT_CORP_ID, WECHAT_AGENT_ID, WECHAT_SECRET, WECHAT_TOKEN
# ⚠️ 多线程访问无锁保护
```

**整改方案**:
```python
import threading

class WeChatConfig:
    _instance = None
    _lock = threading.Lock()
    _config = {}

    @classmethod
    def init(cls):
        with cls._lock:
            cls._config = {
                'corp_id': os.getenv('WECHAT_CORP_ID'),
                'agent_id': os.getenv('WECHAT_AGENT_ID'),
                'secret': os.getenv('WECHAT_SECRET'),
            }

    @classmethod
    def get(cls, key):
        with cls._lock:
            return cls._config.get(key)
```
