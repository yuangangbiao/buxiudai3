## 三、流程引擎

| flow_type | 步数 | 触发 | 流程联动 |
|-----------|------|------|----------|
| `production` | 8 | 桌面/默认 | 报工→质检→入库 |
| `material_purchase` | 5 | 桌面/维修触发 | 到货→入库→领料→出库 |
| `quality` | 4 | 桌面质检单 | 放行→入库 |
| `repair` | 5 | 微信报修 | 维修(记配件)→完工 |
| `outsource` | 8 | 桌面外协 | 回厂→入库 |

### 流程判定

```
1. 请求体 flow_type 字段 → 直接采用
2. product_flow_map 表查询 → 旧兼容
3. 默认 production
```

### production (8步)

```
published → scheduled → confirmed → in_production
    → qc_passed (并行)
    → reported → completed → shipped
```

### material_purchase (5步)

```
material_requested → material_confirmed → material_deadline
    → material_arrived → material_delivered
```

### quality (4步)

```
quality_received → quality_judged → quality_approved → completed
```

### repair (5步)

```
repair_reported → repair_confirmed → repair_in_progress(记配件)
    → repair_verified → completed
```

### outsource (8步)

```
outsource_created → outsource_confirmed → outsource_production
    → outsource_qc → outsource_returned
    → qc_passed (并行) → completed → shipped
```
