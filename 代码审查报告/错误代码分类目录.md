# 不锈钢网带跟单系统 错误代码分类目录

**文档版本**: v1.0
**创建日期**: 2026-05-01
**数据来源**: app.log (2026-04-29) + 代码审查问题
**用途**: 统一错误追踪、快速定位、分类处理

---

## 错误编号体系

### 编号格式
```
ERR-{类别}-{三位序号}
```

### 类别编码

| 编码 | 类别 | 说明 | 严重等级 |
|-----|------|------|---------|
| SYS | 系统错误 | 启动、导入、语法错误 | 🔴 CRITICAL |
| DB | 数据库错误 | 连接、查询、事务错误 | 🔴 CRITICAL |
| NET | 网络错误 | API调用、超时、连接错误 | 🟠 HIGH |
| SEC | 安全错误 | 认证、权限、注入风险 | 🔴 CRITICAL |
| VAL | 验证错误 | 数据校验、类型转换错误 | 🟠 HIGH |
| BIZ | 业务错误 | 逻辑错误、状态不一致 | 🟡 MEDIUM |
| INT | 集成错误 | 模块导入、依赖缺失 | 🔴 CRITICAL |
| RES | 资源错误 | 内存、连接池、文件IO | 🟠 HIGH |

## 错误代码详细目录

### 🔴 SYS 类 - 系统启动错误

| 错误代码 | 错误信息 | 原因 | 解决方案 |
|---------|---------|------|---------|
| ERR-SYS-001 | `Non-UTF-8 code starting with '\xe5' on line N` | 文件保存编码非UTF-8 | 使用VSCode/IDE另存为UTF-8编码 |
| ERR-SYS-002 | `invalid character '�' (U+FFFD)` | 文件含BOM或损坏字符 | 删除文件重新创建或清理BOM |
| ERR-SYS-003 | `SyntaxError: unterminated string literal` | 字符串未正确闭合 | 检查引号配对，特别是中文引号 |
| ERR-SYS-004 | `IndentationError: unexpected indent` | 缩进错误(混用Tab/空格) | 统一使用4空格缩进 |
| ERR-SYS-005 | `IndentationError: unindent does not match` | 缩进级别不匹配 | 检查if/else/for块的缩进 |
| ERR-SYS-006 | `SyntaxError: invalid syntax` | 语法错误(缺少冒号/括号) | 检查语法结构 |
| ERR-SYS-007 | `expected an indented block after 'if' statement` | if语句后缺少代码块 | 添加pass或实际代码 |

  SyntaxError: Non-UTF-8 code starting with '\xe5' on line 3
  File "database.py", line 3
    数据库连接与初始�?
            ^

### 🔴 DB 类 - 数据库错误

| 错误代码 | 错误信息 | 原因 | 解决方案 |
|---------|---------|------|---------|
| ERR-DB-001 | `can't connect to MySQL server` | MySQL服务未启动/防火墙 | 检查MySQL服务状态和端口 |
| ERR-DB-002 | `Access denied for user` | 用户名/密码错误 | 检查db_config.py配置 |
| ERR-DB-003 | `Unknown database 'xxx'` | 数据库不存在 | 执行init_db或检查DB_NAME |
| ERR-DB-004 | `Table 'xxx' doesn't exist` | 表未创建 | 运行数据库迁移脚本 |
| ERR-DB-005 | `Duplicate entry 'xxx' for key 'PRIMARY'` | 主键冲突 | 检查插入逻辑或使用REPLACE |
| ERR-DB-006 | `Lock wait timeout exceeded` | 事务锁等待超时 | 优化大事务或减少锁范围 |
| ERR-DB-007 | `Too many connections` | 连接数超限 | 检查连接池配置和连接泄漏 |
| ERR-DB-008 | `Lost connection to MySQL server` | 连接断开 | 检查网络稳定性或超时设置 |

**关联问题**: P0-7 连接池大小硬编码为5

### 🔴 INT 类 - 模块导入错误

| 错误代码 | 错误信息 | 原因 | 解决方案 |
|---------|---------|------|---------|
| ERR-INT-001 | `No module named 'xxx'` | 模块/包未安装或路径错误 | 检查sys.path和pip安装 |
| ERR-INT-002 | `cannot import name 'xxx' from 'config'` | config.py中未定义该名称 | 检查config.py导出或改名 |
| ERR-INT-003 | `No module named 'constants'` | constants.py文件不存在 | 检查constants.py是否在根目录 |
| ERR-INT-004 | `No module named 'utils'` | utils包未正确初始化 | 检查utils/__init__.py |
| ERR-INT-005 | `name 'xxx' is not defined` | 函数/变量未定义 | 检查定义顺序或拼写错误 |
| ERR-INT-006 | `cannot import name '_get_current_db_type'` | 内部函数位置错误 | 检查函数定义位置 |
