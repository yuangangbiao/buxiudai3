# 错误报错机制设计方案

**项目**: 不锈钢网带跟单系统
**版本**: v1.0
**日期**: 2026-05-01
**性质**: 技术方案（不涉及代码修改）

---

## 一、现状分析

### 1.1 现有异常处理

| 文件 | 现状 |
|-----|------|
| `core/exceptions.py` | 已有 BusinessException 基类，包含 code/message/details |
| `main.py` | 有全局异常处理器 `sys.excepthook`，但无错误编码 |
| `models/database.py` | 有 logger.error 记录，但无统一编码 |
| `app.log` | 记录原始错误信息，无错误编码追踪 |

### 1.2 现有异常类

```python
# 已有结构
class BusinessException(Exception):
    code: str      # 但目前只有通用代码如 'VALIDATION_ERROR'
    message: str
    details: dict
```

### 1.3 缺失能力

- 无标准化错误编码体系
- 错误信息未关联到错误编码文档
- 用户看到错误无法快速定位原因
- 缺乏错误编码查询界面

---

## 二、设计目标

### 2.1 功能目标

1. **统一错误编码**: 所有错误分配标准化编码 `ERR-{类别}-{序号}`
2. **自动关联文档**: 错误发生时自动提示错误编码，用户可查阅文档
3. **快速定位**: 根据错误编码30秒内定位问题原因和解决方案
4. **历史追踪**: 错误日志包含编码，便于统计分析

### 2.2 非功能目标

- 不破坏现有业务逻辑
- 最小化性能影响
- 向后兼容现有异常处理

---

## 三、错误编码体系

### 3.1 编码格式

```
ERR-{类别}-{三位序号}
```

**示例**: `ERR-SYS-001`, `ERR-DB-002`, `ERR-BIZ-003`

### 3.2 类别定义

| 类别 | 前缀 | 说明 | 严重等级 |
|-----|------|------|---------|
| SYS | ERR-SYS | 系统语法/启动错误 | 🔴 CRITICAL |
| DB | ERR-DB | 数据库错误 | 🔴 CRITICAL |
| INT | ERR-INT | 模块导入错误 | 🔴 CRITICAL |
| SEC | ERR-SEC | 安全错误 | 🔴 CRITICAL |
| NET | ERR-NET | 网络错误 | 🟠 HIGH |
| VAL | ERR-VAL | 数据验证错误 | 🟠 HIGH |
| BIZ | ERR-BIZ | 业务逻辑错误 | 🟡 MEDIUM |
| RES | ERR-RES | 资源管理错误 | 🟠 HIGH |

    """根据错误编码获取错误信息"""
    return ERROR_CODES.get(code)

def format_error_for_display(code: str, original_error: str = None) -> str:
    """格式化错误信息用于显示给用户"""
    error_info = get_error_info(code)
    if not error_info:
        return f"未知错误编码: {code}\n原始错误: {original_error}"

    msg = f"""
╔══════════════════════════════════════════════════════════╗
║  错误编码: {code:<44}║
╠══════════════════════════════════════════════════════════╣
║  错误名称: {error_info.name:<43}║
║  严重程度: {error_info.severity:<43}║
╠══════════════════════════════════════════════════════════╣
║  错误描述:                                                  ║
║  {error_info.message:<53}║
╠══════════════════════════════════════════════════════════╣
║  原因分析:                                                  ║
║  {error_info.cause:<53}║
╠══════════════════════════════════════════════════════════╣
║  解决方案:                                                  ║
"""
    for line in error_info.solution.split('\n'):
        msg += f"║  {line:<53}║"

    msg += """
╠══════════════════════════════════════════════════════════╣
║  错误示例:                                                  ║
"""
    if error_info.example:
        msg += f"║  {error_info.example:<53}║"

    msg += """
╠══════════════════════════════════════════════════════════╣
║  查看详细文档: 代码审查报告/错误代码分类目录.md           ║
╚══════════════════════════════════════════════════════════╝
"""
    return msg

### 4.2 创建错误处理器模块

**文件**: `core/error_handler.py`

```python
# -*- coding: utf-8 -*-
"""
统一错误处理器
负责错误编码识别、日志记录和用户提示
"""

import logging
import traceback
import re
from typing import Optional, Tuple
from core.error_codes import ERROR_CODES, get_error_info, format_error_for_display

logger = logging.getLogger(__name__)

# 错误模式匹配表：正则表达式 -> 错误编码
ERROR_PATTERNS = [
    # SYS 类
    (r"Non-UTF-8 code starting with", "ERR-SYS-001"),
    (r"invalid character.*U\+FFFD", "ERR-SYS-002"),
    (r"unterminated string literal", "ERR-SYS-003"),
    (r"IndentationError", "ERR-SYS-004"),
    (r"SyntaxError.*invalid syntax", "ERR-SYS-006"),

    # DB 类
    (r"Can't connect to MySQL", "ERR-DB-001"),
    (r"Access denied.*password", "ERR-DB-002"),
    (r"Unknown database", "ERR-DB-003"),
    (r"Lost connection to MySQL", "ERR-DB-008"),
    (r"BLOB.*can't have a default value", "ERR-DB-009"),

    # INT 类
    (r"No module named", "ERR-INT-001"),
    (r"cannot import name", "ERR-INT-002"),
    (r"name.*is not defined", "ERR-INT-005"),

    # SEC 类
    (r"hard-coded.*password", "ERR-SEC-001"),

    # VAL 类
    (r"invalid literal for.*float", "ERR-VAL-001"),
    (r"invalid literal for.*int", "ERR-VAL-002"),
    (r"JSONDecodeError", "ERR-VAL-003"),

    # RES 类
    (r"cursor is closed", "ERR-RES-001"),
    (r"pool exhausted|Too many connections", "ERR-DB-007"),
]


def recognize_error_code(error_msg: str) -> Optional[str]:
    """
    从错误信息中识别错误编码

    Args:
        error_msg: 原始错误信息

    Returns:
        错误编码，如 "ERR-SYS-001"，未识别返回 None
    """
    for pattern, code in ERROR_PATTERNS:
        if re.search(pattern, error_msg, re.IGNORECASE):
            return code
    return None


def handle_error(exc_type: type, exc_value: Exception, exc_traceback) -> Tuple[str, Optional[str]]:
    """
    处理错误，自动识别编码并记录

    Args:
        exc_type: 异常类型
        exc_value: 异常值
        exc_traceback: 堆栈跟踪

    Returns:
        (错误编码, 格式化后的用户提示)
    """
    error_msg = str(exc_value)
    traceback_str = ''.join(traceback.format_exception(exc_type, exc_value, exc_traceback))

    # 1. 识别错误编码
    error_code = recognize_error_code(error_msg)

    # 2. 记录日志（包含编码）
    if error_code:
        logger.error(f"[{error_code}] {error_msg}")
        logger.error(f"[{error_code}] 完整堆栈:\n{traceback_str}")
    else:
        logger.error(f"[UNKNOWN] {error_msg}")
        logger.error(f"[UNKNOWN] 完整堆栈:\n{traceback_str}")

    # 3. 生成用户提示
    user_hint = format_error_for_display(error_code or "UNKNOWN", error_msg)

    return error_code, user_hint


def show_error_dialog(error_code: str, error_msg: str):
    """
    显示错误对话框（当图形界面可用时）

    Args:
        error_code: 错误编码
        error_msg: 原始错误信息
    """
    hint = format_error_for_display(error_code, error_msg)

    try:
        from tkinter import messagebox
        messagebox.showerror(f"错误 {error_code}", hint)
    except:
        print(hint)


def log_to_db(error_code: str, error_msg: str, traceback_str: str, context: dict = None):
    """
    将错误记录到数据库（可选）

    Args:
        error_code: 错误编码
        error_msg: 错误信息
        traceback_str: 堆栈跟踪
        context: 额外上下文信息
    """
    try:
        from models.database import get_connection
        conn = get_connection()
        cursor = conn.cursor()

        cursor.execute("""
            INSERT INTO error_logs (error_code, error_message, traceback, context, created_at)
            VALUES (%s, %s, %s, %s, NOW())
        """, (error_code, error_msg, traceback_str, str(context or {})))

        conn.commit()
        cursor.close()
        conn.close()
    except Exception as e:
        logger.warning(f"记录错误到数据库失败: {e}")
```

### 4.3 修改 main.py 全局异常处理器

**文件**: `main.py` (修改部分)

```python
# 在文件开头添加导入
from core.error_handler import recognize_error_code, handle_error, show_error_dialog

# 修改全局异常处理器
def log_error(exc_type, exc_value, exc_traceback):
    """全局异常处理器"""
    if issubclass(exc_type, KeyboardInterrupt):
        sys.__excepthook__(exc_type, exc_value, exc_traceback)
        return

    # 使用统一错误处理器
    error_code, user_hint = handle_error(exc_type, exc_value, exc_traceback)

    # 显示错误对话框
    show_error_dialog(error_code or "UNKNOWN", str(exc_value))

# 设置全局异常处理器
sys.excepthook = log_error
```

### 4.4 创建错误查询界面（可选）

**文件**: `views/error_lookup_view.py`

```python
# -*- coding: utf-8 -*-
"""
错误编码查询界面
用户可通过此界面根据错误编码查询解决方案
"""

from tkinter import ttk, messagebox

class ErrorLookupView:
    """错误编码查询对话框"""

    def __init__(self, parent):
        self.parent = parent
        self.create_dialog()

    def create_dialog(self):
        """创建查询对话框"""
        from core.error_codes import ERROR_CODES

        dialog = Toplevel(self.parent)
        dialog.title("错误编码查询")
        dialog.geometry("800x600")

        # 输入框
        ttk.Label(dialog, text="请输入错误编码 (如: ERR-SYS-001):").pack(pady=5)
        self.code_entry = ttk.Entry(dialog, width=30)
        self.code_entry.pack(pady=5)
        ttk.Button(dialog, text="查询", command=self.lookup).pack(pady=5)

        # 结果显示
        self.result_text = tk.Text(dialog, width=90, height=25)
        self.result_text.pack(pady=10, padx=10)

        # 填充所有编码列表
        ttk.Label(dialog, text="常见错误编码:").pack(pady=5)
        for code in list(ERROR_CODES.keys())[:10]:
            ttk.Label(dialog, text=code, foreground="blue").pack()

    def lookup(self):
        """执行查询"""
        from core.error_codes import format_error_for_display, get_error_info

        code = self.code_entry.get().strip().upper()
        if not code:
            return

        error_info = get_error_info(code)
        if error_info:
            result = format_error_for_display(code)
            self.result_text.delete(1.0, END)
            self.result_text.insert(1.0, result)
        else:
            messagebox.showwarning("未找到", f"错误编码 {code} 不存在")
```

-- 错误日志表
CREATE TABLE IF NOT EXISTS error_logs (
    id INTEGER PRIMARY KEY AUTO_INCREMENT,
    error_code VARCHAR(20) NOT NULL,          -- 错误编码
    error_message TEXT NOT NULL,              -- 原始错误信息
    traceback TEXT,                           -- 堆栈跟踪

    created_at DATETIME NOT NULL,             -- 发生时间
    resolved_at DATETIME,                     -- 解决时间
    resolution TEXT                          -- 解决方法记录
);
