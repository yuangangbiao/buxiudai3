# 消息模板引擎全面打通方案 v3.0

> 版本: v3.0 | 日期: 2026-06-02 | 状态: 待执行

## 一、现状诊断

v2.x 实现后存在 3 条通路各走各的问题:

```
通路1: template_engine._render_template → MESSAGE_TEMPLATES_DEFAULT (不读 MySQL)
通路2: dispatch_center GET/POST /templates → _dispatch_cache (旧缓存)
通路3: dispatch_center PUT/DELETE /templates → MySQL (新, 已实现)
```

结果: 自定义模板写入了 MySQL 但 `_render_template` 不读, 渲染时用不上。

## 二、修复目标

打通 `_render_template` → MySQL → 渲染 → 发送 全链路:

```
任何调用方 → _render_template(tid, vars)
    ↓
1. 先查 MySQL.message_templates (is_active=1)
    ↓ 有 → 用 MySQL 版本
    ↓ 无 ↓
2. 降级 MESSAGE_TEMPLATES_DEFAULT
    ↓
3. 变量替换 (None→空, 未匹配→—)
    ↓
4. 返回渲染后的 content
```

**现状**:

**改为**:
```python
def _render_template(template_id: str, variables: dict) -> str:
    content = None
    # 1. 优先从 MySQL 加载自定义模板
    try:
        import pymysql
        c = pymysql.connect(
            host='127.0.0.1', port=3306, user='root', password='88888888',
            database='container_center', charset='utf8mb4',
            connect_timeout=2
        )
        cur = c.cursor()
        cur.execute('SELECT content FROM message_templates WHERE id=%s AND is_active=1', (template_id,))
        row = cur.fetchone()
        c.close()
        if row:
            content = row[0]
    except Exception as e:
        logger.debug("MySQL 模板查询失败, 降级内置: %s", e)
    # 2. MySQL 无 → 降级内置
    if not content:
        template = next((t for t in MESSAGE_TEMPLATES_DEFAULT if t.get('id') == template_id), None)
        if not template:
            logger.warning("模板 %s 不存在", template_id)
            return ''
        content = template['content']
    ...  # 后续变量替换不变
```

### 改动 2: dispatch_center.py — 统一 GET/POST 数据源

**文件**: `mobile_api_ai/dispatch_center.py`
**位置**: `get_templates` 函数 (行 1465) 和 `save_template` 函数 (行 1473)

**现状**:
```python
@dispatch_center_bp.route('/templates', methods=['GET'])
def get_templates():
    data = _dispatch_cache.get_data()
    templates = data.get('templates', [])
    current_template = data.get('message_template', '')
    return jsonify({'code': 0, 'data': {'templates': templates, 'current': current_template}})
```

**改为**: 合并内置 + MySQL 自定义:
```python
@dispatch_center_bp.route('/templates', methods=['GET'])
def get_templates():
    builtin = [{
        'id': t['id'], 'name': t['name'], 'category': t['category'],
        'title': t.get('title', ''), 'content_preview': t['content'][:60],
        'is_builtin': True
    } for t in MESSAGE_TEMPLATES_DEFAULT]
    try:
        import pymysql
        c = pymysql.connect(host='127.0.0.1', port=3306, user='root', password='88888888',
                           database='container_center', charset='utf8mb4')
        cur = c.cursor()
        cur.execute('SELECT id, name, category, content, version, is_builtin FROM message_templates WHERE is_active=1')
        custom = [{
            'id': r[0], 'name': r[1], 'category': r[2],
            'content_preview': r[3][:60], 'version': r[4], 'is_builtin': bool(r[5])
        } for r in cur.fetchall()]
        c.close()
    except:
        custom = []
    return jsonify({'code': 0, 'data': {'builtin': builtin, 'custom': custom, 'total': len(builtin) + len(custom)}})
```

**废弃旧 POST** (写入 _dispatch_cache):
```python
@dispatch_center_bp.route('/templates', methods=['POST'])
```
