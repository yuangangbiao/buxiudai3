## 🚀 启动方式

### 方式一：使用启动脚本（推荐）
```bash
cd d:\yuan\不锈钢网带跟单3.0
python start_services.py
```

### 方式二：手动启动单个服务
```bash
# 容器中心 API
cd mobile_api_ai
python container_center_api.py

# 调度中心（含企业微信机器人）
cd mobile_api_ai
python standalone_dispatch_server.py

# 大屏服务器
cd desktop/views/dashboard
python dashboard_server.py

# 轮询监控器
cd mobile_api_ai
python poll_monitor.py
```

---

## 📝 注意事项

1. **容器中心 API（5002）和调度中心（5003）必须同时运行**，两者互相依赖
2. **大屏服务器需要 MySQL 数据库**，请确保数据库配置正确
3. **企业微信机器人功能已整合到 standalone_dispatch_server.py**，无需单独启动
4. **只有大屏服务器有可视化前端页面**
5. **移动端工人报工 API** 在 `standalone_dispatch_server.py` 的 `/api/sync/report` 端点
