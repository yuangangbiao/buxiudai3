# 服务访问说明

## 📋 服务列表

| 服务名称 | 端口 | 访问地址 | 说明 |
|---------|------|---------|------|
| **移动报工API** | 5000 | http://localhost:5000 | 不锈钢跟单系统主API服务，返回JSON |
| **企业微信机器人** | 5003 | http://localhost:5003 | API服务，无前端页面 |
| **大屏服务器** | 5005 | http://localhost:5005 | 有前端页面！ |
| **轮询监控器** | - | - | 后台服务 |

---

## 🌐 推荐访问页面

### 1. 数据大屏 (推荐)
**地址**: http://localhost:5005

**可选版本**:
- http://localhost:5005/ - 方案2 (默认，进度卡片版)
- http://localhost:5005/v1 - 方案1 (深蓝经典版)
- http://localhost:5005/v2 - 方案2 (进度卡片版)
- http://localhost:5005/v3 - 方案3 (紧凑卡片版)

### 2. API健康检查
- 移动报工API: http://localhost:5000/health

---

## 🚀 启动方式

### 方式一：使用启动脚本 (推荐)
```bash
cd d:\yuan\不锈钢网带跟单3.0
python start_services.py
```

### 方式二：手动启动单个服务
```bash
# 移动报工API (不锈钢跟单系统主服务)
cd mobile_api_ai
python app.py

# 企业微信机器人
cd mobile_api_ai
python wechat_server.py

# 大屏服务器
cd views/dashboard
python dashboard_server.py

# 轮询监控器
cd mobile_api_ai
python poll_monitor.py
```

---

## 📝 注意事项

1. **大屏服务器需要MySQL数据库**，请确保数据库配置正确
2. **企业微信机器人需要Redis**，可选，不影响核心功能
3. **移动报工API是纯API服务**，返回JSON数据，没有前端页面
4. **只有大屏服务器有可视化前端页面**
5. **主软件是不锈钢自动跟单系统3.0版**，请使用端口5000访问移动报工API
