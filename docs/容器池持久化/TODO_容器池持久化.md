### 3.2 日志关键词

在日志中搜索以下关键词可监控持久化状态：

```
[TaskPool] 存储初始化成功
[TaskPool] 存储初始化失败
[TaskPool] 保存任务失败
[TaskPool] 从存储加载失败
```

## 四、注意事项

| 事项 | 说明 |
|------|------|
| 服务重启 | 重启后任务自动从数据库恢复 |
| 异常降级 | 存储失败时自动降级到内存（服务继续运行但数据不持久化） |
| 数据库迁移 | 如需迁移数据库，复制 `task_pool.db` 文件即可 |
