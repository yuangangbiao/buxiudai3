1. **5001 桌面 Web 只复刻了 1 个视图(订单列表)**,剩余 26 个 Tkinter 视图(生产/物料/质检/排产/报修/外协/报表/看板/规则/...) 待 Web 化 — 需小圣按 P1 路线图 8 周交付

4. **鉴权 token 无过期机制** — 永不过期,泄露即风险。需下轮加 JWT 或 session 过期
5. **`/api/dispatch-center/login` 页面有 SQL/HTML 注入风险?** — 登录端点只用 username 字段,参数化查询安全 ✅
6. **云端 124.223.57.82:5006 直连代码未清理** — standalone_dispatch_server.py:1012 / scripts/cloud/* 多处仍直连,违反云端通信架构规范,待办
