## 创建企业微信应用

### 第一步：登录管理后台

访问：https://work.weixin.qq.com/wework_admin/

### 第二步：创建应用

1. 点击 **"应用管理"** → **"应用"**
2. 点击 **"创建应用"**

填写应用信息：
```
应用名称：生产任务助手
应用简介：任务分配、扫码报工
可见范围：选择需要使用的部门或成员
```

### 第三步：获取凭证

创建成功后，在应用详情页：

| 凭证 | 位置 | 保存到 .env 变量 |
|------|------|-----------------|
| CorpID | 我的企业 → 企业ID | `WECHAT_CORP_ID` |
| AgentID | 应用详情页 | `WECHAT_AGENT_ID` |
| Secret | 点击"获取" → 只显示一次 | `WECHAT_SECRET` |

⚠️ **重要**：Secret 只显示一次，请立即复制保存！

## 配置回调接口

### 第一步：配置接收消息

在应用详情页找到 **"接收消息"** 区域：

| 配置项 | 值 |
|--------|-----|
| **URL** | `https://your-domain.com/api/wechat/app/hook` |
| **Token** | 自己设置（比如：`mytaskbot2026`） |
| **EncodingAESKey** | 点击"随机生成" |

**注意**：
- URL 必须是 HTTPS
- Token 和 EncodingAESKey 保存到 `.env` 文件

### 第二步：先不要点保存

先进行服务器部署，等服务器运行后再点击"保存"进行验证。

## 服务器部署

### 方案A：本地开发 + 内网穿透

#### 1. 配置 .env 文件

复制 `.env.example` 为 `.env` 并填写：

```bash
cd mobile_api_ai
copy .env.example .env
# 编辑 .env 填写配置
```

#### 2. 启动内网穿透（以 ngrok 为例）

```bash
# 下载 ngrok 后运行
ngrok http 5003
```

会显示公网地址：
```
Forwarding  https://xxxx-xx-xx-xx.ngrok-free.app -> http://localhost:5003
```

#### 3. 启动应用

```bash
python wechat_work_bot_v2.py
```

#### 4. 在企业微信后台填写回调 URL

```
https://xxxx-xx-xx-xx.ngrok-free.app/api/wechat/app/hook
```

点击 **"保存"**，如果成功会提示验证通过。

---

### 方案B：云服务器部署

#### 1. 购买云服务器

推荐：
- 阿里云、腾讯云、华为云
- 配置：1核2G 够用

#### 2. 配置安全组

开放端口：
- 80 (HTTP)
- 443 (HTTPS)
- 5003 (可选，应用内部端口)

#### 3. 部署代码

```bash
# 上传代码到服务器
# 安装依赖
pip install pycryptodome requests python-dotenv flask

# 配置 .env
cd mobile_api_ai
cp .env.example .env
vim .env  # 编辑配置

# 启动
python wechat_work_bot_v2.py
```

#### 4. 配置 HTTPS

使用 Nginx + Let's Encrypt 配置 HTTPS（略）

## 常见问题

### Q1: URL验证失败？

检查：
1. 服务器是否正常运行
2. HTTPS 是否配置正确
3. Token 是否一致
4. 查看服务器日志

### Q2: 收不到消息？

检查：
1. 应用的"可见范围"是否包含你
2. 回调URL是否正确
3. 查看服务器日志

### Q3: 如何配置权限？

在企业微信后台：
- 应用管理 → 应用 → 可见范围
- 选择允许使用的部门或成员
