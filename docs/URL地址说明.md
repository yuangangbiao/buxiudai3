# URL地址说明

## 端口映射

| 端口 | 服务 | 启动文件 |
|------|------|----------|
| 5002 | 容器中心 API | container_center_api.py |
| 5003 | 调度中心 + 企业微信 | standalone_dispatch_server.py |

## 完整URL清单

### :5003 (调度中心主服务)

| URL | 页面 | 说明 |
|-----|------|------|
| `http://192.168.0.104:5003/` | 调度中心主页 | 工单管理、流程编排 |
| `http://192.168.0.104:5003/container/` | 容器仪表盘 | 容器监控数据可视化 |
| `http://192.168.0.104:5003/container/config` | 容器配置管理 | 节点、存储桶配置 |

### :5002 (容器中心API)

| URL | 页面 | 说明 |
|-----|------|------|
| `http://192.168.0.104:5002/` | 容器中心API状态 | JSON状态信息 |
| `http://192.168.0.104:5002/container/` | 容器仪表盘 | 与5003共享相同页面 |
