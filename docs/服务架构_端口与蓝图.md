# 服务端口与蓝图映射

## 端口架构

| 端口 | 服务文件 | 蓝图 | 用途 |
|------|---------|------|------|
| **5003** | `standalone_dispatch_server.py` | dispatch_center | 调度中心核心逻辑 |
| **5008** | `app.py` | process, scan, quality_inspection 等 | 移动端 API（工序/扫码/质检等） |
| 5006 | 云端（`124.223.57.82`） | - | 微信消息中转 |

## 关键教训

- 调试 API 时先确认该端点注册在哪个服务（5003 还是 5008）
- 重启 5003 → standalone_dispatch_server.py
- 重启 5008 → app.py
- `wechat_server.py` 是 5008 的旧入口，不要覆盖
