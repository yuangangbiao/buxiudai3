## 二、修复方案

### 问题 1: localhost 白名单绕过

**文件**: `mobile_api_ai/container_center_api.py`
**位置**: 第 2039 行

**当前代码**:
```python
_MIRROR_WHITELIST_IPS = frozenset(['127.0.0.1', '::1', 'localhost'])
```

**修复代码**:
```python
import os

# 生产环境禁用白名单
if os.getenv('FLASK_ENV') == 'production':
    _MIRROR_WHITELIST_IPS = frozenset()
else:
    _MIRROR_WHITELIST_IPS = frozenset(['127.0.0.1', '::1', 'localhost'])
```

**验证命令**:
```python
# 开发环境
os.environ['FLASK_ENV'] = 'development'
# 白名单应包含 localhost

# 生产环境
os.environ['FLASK_ENV'] = 'production'
# 白名单应为空
```

---

### 问题 2: 硬编码路径

**文件**: `mobile_api_ai/storage/mysql_storage.py`
**位置**: 第 12 行

**当前代码**:
```python
load_dotenv(r'D:\yuan\不锈钢网带跟单3.0\.env', override=True)
```

**修复代码**:
```python
import os
from pathlib import Path

# 动态计算项目根目录
PROJECT_ROOT = Path(__file__).resolve().parent.parent.parent
ENV_FILE = PROJECT_ROOT / '.env'

load_dotenv(str(ENV_FILE), override=True)
```

**验证命令**:
```python
from storage.mysql_storage import ENV_FILE
print(f"Env file exists: {ENV_FILE.exists()}")
# 应输出: True
```
