## 教训

1. **永远独立验证**：完成时必须用 grep 验证声称的状态
2. **不要相信工具返回的 success**：实际可能是失败但工具报告成功
3. **文档后写**：先改代码，验证完成后再写文档
4. **跨版本对比**：每次完成时对照上一版的状态变化
