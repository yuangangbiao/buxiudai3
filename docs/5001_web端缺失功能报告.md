## 五、技术说明

- Web端和桌面端**共用同一个 MySQL 数据库**（`core.db.get_connection`），数据层完全对等
- Web端操作员管理已重构为**企业架构模式**（`enterprise_structure.json` + MySQL 双写），与桌面端逻辑不同
- 订单录入/查询/导出功能 web5001 **完全可替代**桌面端
