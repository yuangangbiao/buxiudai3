## 8. 业务流端到端覆盖

按真实下单到发货全流程：

```
新建订单 → 工艺计算 → 排产调度 → 工单发布
    → 工人扫码报工 → 工序进度 → 质检 → 包装
    → 成品入库 → 发货 → 物流跟踪
    → 财务成本核算 → 报表统计
```

每一步都涉及：
- **桌面端**：下单 / 计算 / 调度 / 看板
- **Web 5001**：管理 / 配置 / 报表
- **Mobile 5008**：扫码 / 报工 / 拍照 / 查询
- **5003 调度**：工单 / 排产 / 容器
- **8008 同步**：跨服务数据同步
- **云端微信**：通知推送
