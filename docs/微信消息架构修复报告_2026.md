#### 问题 2: 超时配置不统一

**位置**: 20+ 处使用不同的超时配置

**当前问题**:
```python
# 混乱的超时配置
timeout=int(os.environ.get('REQUEST_TIMEOUT_FAST', '5'))      # 方式 1
timeout=REQUEST_TIMEOUT_NORMAL                                # 方式 2
timeout=10                                                    # 方式 3：硬编码
timeout=15                                                    # 方式 4：硬编码
```

**修复方案**:
```python
# 统一使用 REQUEST_TIMEOUT_* 常量
from core.config import REQUEST_TIMEOUT_FAST, REQUEST_TIMEOUT_NORMAL, REQUEST_TIMEOUT_LONG

# 快速操作
timeout=REQUEST_TIMEOUT_FAST

# 正常操作
timeout=REQUEST_TIMEOUT_NORMAL

# 长时间操作
timeout=REQUEST_TIMEOUT_LONG
```

#### 问题 3: AppBotProxy 未真正阻止直接调用

**位置**: `mobile_api_ai/wechat_work_bot_bp.py:495-511`

**当前状态**:
```python
def send_text(self, text):
    _violation_forbidden_bot('UserBot.AppBotProxy.send_text',
        'AppBotProxy.send_text 直接调用，应改用 wechat_msg_dispatcher.send_templated()')
    from wechat_app_bot import get_app_bot
    app_bot = get_app_bot()
    if app_bot:
        app_bot.send_text_to_user(self.target_user, text)
```

**问题**: 有警告但仍执行发送，未真正阻止

**修复方案**:
```python
# 方案 A: 真正阻止（抛出异常）
def send_text(self, text):
    _violation_forbidden_bot('UserBot.AppBotProxy.send_text',
        'AppBotProxy.send_text 已禁用，请改用 wechat_msg_dispatcher.send_templated()')
    raise NotImplementedError('AppBotProxy.send_text 已禁用')

# 方案 B: 自动路由到统一入口
def send_text(self, text):
    from wechat_msg_dispatcher import send_templated
    send_templated('custom_message', {'content': text},
                   force_receivers=[self.target_user])
```

## 三、接收人统一管理

### 3.1 管理架构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          接收人统一管理架构                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  【数据库层】                                                             │
│  └── steel_belt.notification_recipient_preset 表                          │
│      ├── scenario    VARCHAR(128)  触发场景                              │
│      ├── receivers   JSON           接收人列表 ["张三","李四"]           │
│      ├── enabled     TINYINT        1=启用 0=禁用                        │
│      └── force_by_assignee TINYINT 1=强制按执行人发送                    │
│                                    ↓                                        │
│  【后端服务层】                                                          │
│  └── notification_preset_service.py                                       │
│      ├── get_receivers_for_scenario()    查询接收人                       │
│      ├── set_receivers_for_scenario()    设置接收人                       │
│      ├── delete_preset()                  删除预设                         │
│      └── list_all_presets()              列出所有预设                     │
│                                    ↓                                        │
│  【REST API 层】                                                         │
│  └── dispatch_center/_core.py:1906-1962                                 │
│      ├── GET  /notification/presets           查询所有                    │
│      ├── PUT  /notification/presets/<scenario> 更新                      │
│      ├── DELETE /notification/presets/<scenario> 删除                    │
│      └── POST /notification/presets/<scenario>/toggle 启用/禁用           │
│                                    ↓                                        │
│  【前端配置层】                                                          │
│  ├── 调度中心 (dispatch_center.html)                                     │
│  │   └── 消息模板 → 编辑模板 → 接收人设置                              │
│  │       ├── 按任务执行人（force_by_assignee）                          │
│  │       └── 部门/角色/操作员选择                                       │
│  └── 配置中心 (config_center.html)                                       │
│      └── 通知配置 → 全局通知开关                                        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3.2 前端管理位置

| 页面 | 路径 | 功能 |
|------|------|------|
| **调度中心 - 消息模板** | `templates/dispatch_center.html:148-167` | 消息模板管理 |
| **调度中心 - 模板编辑** | `templates/dispatch_center.html:698-803` | 接收人设置 |
| **配置中心 - 工序管理** | `templates/config_center.html:480-505` | 工序配置 |
| **配置中心 - 通知配置** | `templates/config_center.html:507-559` | 全局通知开关 |

### 3.3 接收人获取优先级

```
1. 调用 send_templated(scenario='workorder_complete')
           ↓
2. wechat_msg_dispatcher.send_templated()
           ↓
3. 优先查询数据库: NotificationPresetService.get_receivers_for_scenario()
   ├── 存在且启用 → 返回 receivers
   └── 不存在 → 回退到代码默认值
           ↓
4. 返回接收人列表给消息发送函数
```
