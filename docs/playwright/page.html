📋 使用说明

1. 云端服务器部署

在云端服务器运行 python wechat_cloud.py

2. 配置云端地址

将云端服务器的域名或IP填入上方"云端地址"，并填写正确的API密钥

3. 启用连接

勾选"启用云端连接"后保存，本地服务将自动开始轮询云端

4. 测试连接

点击"测试连接"按钮验证云端是否可达
