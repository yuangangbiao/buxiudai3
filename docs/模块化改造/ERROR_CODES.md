## 二、详细错误码

### 2.1 通用错误（0-999）

| 错误码 | 消息 | 场景 |
|--------|------|------|
| 0 | success | 成功 |
| 404 | 资源不存在 | URL 不存在 |
| 405 | 方法不允许 | HTTP 方法错误 |

### 2.2 参数错误（1000-1099）

| 错误码 | 消息 | 场景 |
|--------|------|------|
| 1001 | 参数不完整 | 缺少必填字段 |
| 1002 | 参数类型错误 | 字段类型不匹配 |
| 1003 | 参数值非法 | 值超出范围 |
| 1004 | 参数格式错误 | 邮箱/手机号格式错误 |

### 2.3 鉴权错误（1100-1199）

| 错误码 | 消息 | 场景 |
|--------|------|------|
| 1101 | 未授权 | 未登录 |
| 1102 | Token 过期 | Token 已过期 |
| 1103 | Token 无效 | Token 错误 |
| 1104 | 权限不足 | 没有权限 |

### 2.4 请求错误（1200-1299）

| 错误码 | 消息 | 场景 |
|--------|------|------|
| 1201 | JSON 解析失败 | 请求体不是合法 JSON |
| 1202 | 请求体过大 | 超过限制 |
| 1203 | 编码错误 | 字符编码不支持 |
| 1204 | Content-Type 错误 | 非 application/json |

### 2.5 资源错误（1300-1399）

| 错误码 | 消息 | 场景 |
|--------|------|------|
| 1301 | 订单不存在 | 订单号无效 |
| 1302 | 工序不存在 | 工序名无效 |
| 1303 | 用户不存在 | 用户 ID 无效 |
| 1304 | 设备不存在 | 设备名无效 |
| 1305 | 物料不存在 | 物料号无效 |
| 1306 | 容器不存在 | 容器 ID 无效 |

### 2.6 路由错误（1400-1499）

| 错误码 | 消息 | 场景 |
|--------|------|------|
| 1401 | 未授权访问 | - |
| 1403 | 禁止访问 | 权限不足 |
| 1404 | 接口不存在 | URL 错误 |
| 1405 | 方法不允许 | HTTP 方法错误 |

### 2.7 业务错误（1500-1599）

| 错误码 | 消息 | 场景 |
|--------|------|------|
| 1501 | 报工失败 | 报工保存失败 |
| 1502 | 订单已存在 | 创建重复订单 |
| 1503 | 状态非法 | 状态机错误 |
| 1504 | 数量超限 | 报工数量超额 |
| 1505 | 工序已派发 | 重复派发 |
| 1506 | 容器已占用 | 容器被其他订单使用 |

### 2.8 同步错误（1600-1699）

| 错误码 | 消息 | 场景 |
|--------|------|------|
| 1601 | 同步到 steel_belt 失败 | 8008 桥接失败 |
| 1602 | 同步超时 | 8008 响应超时 |
| 1603 | 同步参数错误 | 字段不符合白名单 |
| 1604 | 同步已入队列 | 降级队列记录 |
| 1605 | 同步版本冲突 | 乐观锁失败 |
| 1606 | 跨库直查违规 | 检测到新跨库直查 |

### 2.9 服务器错误（9000-9999）⚠️ 修订

| 错误码 | 消息 | 场景 |
|--------|------|------|
| **9000** | **内部错误** | 通用服务器错误 |
| 9001 | 数据库错误 | DB 连接失败 |
| 9002 | 缓存错误 | Redis 连接失败 |
| 9003 | 第三方服务错误 | 企业微信/钉钉不可用 |
| 9004 | 限流触发 | 请求频率超限 |
| 9005 | 服务降级中 | 系统降级模式 |
| 9006 | 服务维护中 | 计划维护 |

## 三、避免的段位

| 段位 | 用途 | **避开** |
|------|------|----------|
| 5000-5099 | **5002/5003/5008/5010 端口冲突**（容器/调度/报工/库存） | ❌ 不可用 |
| 5001 | 错误码（非服务端口，可作为错误码使用） | ⚠️ 可用但建议避开（避免混淆） |
| 5010 | 端口号（库存管理） | ❌ 不可用 |
| 8008 | 端口号（Sync Bridge） | ❌ 不可用 |

}
