## 五、工厂用户视角：web5001 无法完成的场景

### 场景1：配置质检规则
**问题描述**：工厂需要根据产品类型配置不同的质检项目和判定标准

**桌面端操作**：
1. 打开"质检规则"窗口
2. 新建质检规则（规则名称、质检项目、判定标准）
3. 关联到产品类型
4. 保存

**web5001**：❌ 无法完成，需要联系开发人员修改代码

---

### 场景2：管理BOM物料清单
**问题描述**：新产品需要建立BOM物料清单，以便后续物料备料

**桌面端操作**：
1. 打开"BOM管理"窗口
2. 新建BOM模板
3. 添加物料明细（物料名称、规格、用量）
4. 保存并关联到产品

**web5001**：❌ 无法完成，无法管理产品BOM

---

### 场景3：员工工时绩效考核
**问题描述**：月底需要统计每个员工的工时和报工数量

**桌面端操作**：
1. 打开"操作员管理"窗口
2. 选择要统计的员工
3. 点击"工时统计"
4. 选择月份，即可看到工时、报工数、合格率等数据

**web5001**：
- ⚠️ 需要进入"操作员管理" → 找到员工 → 点击"📊统计"按钮
- 入口较深，流程不够直观

---

### 场景4：数据备份与恢复
**问题描述**：系统升级前需要备份数据库，防止出现问题

**桌面端操作**：
1. 打开"备份管理"窗口
2. 点击"备份数据库"
3. 选择备份位置
4. 开始备份

**web5001**：❌ 无法完成，无备份功能

---

### 场景5：快速查看系统运行日志
**问题描述**：系统出现异常，需要查看运行日志定位问题

**桌面端操作**：
1. 打开"日志查看"窗口
2. 选择日志类型（系统/错误/操作）
3. 设置时间范围
4. 查看日志详情

**web5001**：❌ 无法完成，需要登录服务器查看日志文件
